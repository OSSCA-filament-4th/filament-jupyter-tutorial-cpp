# filament-jupyter-tutorial-cpp

----------------------------
## 단기목표
filament cpp를 jupyter notebook에 띄우기

## INSTALL

### Only possible in Linux and OS X
#### dependency (currently)
`xeus-cling` `jupyter` `notebook`

##### conda
```cmd
conda create -n oss xeus-cling
pip3 install jupyter
```

## RUN
```cmd
jupyter notebook
```
